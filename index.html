About Python + JS vs Python

You already know programming ideas. Python writes them differently.

About Python

Python is the default language for a lot of AI work.

It is readable and widely used for data, AI, and machine learning.

Python
topic = "Python Foundations"
print("Today we are learning:", topic)
Today is not about mastering Python. It is about becoming comfortable enough to continue.
JS vs Python

Code blocks

JavaScript uses curly braces. Python uses colons and indentation.

JavaScript
if (score >= 60) {
  console.log("Pass");
}
Python
score = 72

if score >= 60:
    print("Pass")
JS vs Python

Variable declaration

Python does not use let, const, or var.

JavaScript
let name = "Ravi";
const score = 82;
var passed = true;
Python
name = "Ravi"
score = 82
passed = True

print(name, score, passed)

Printing output

print() is Python’s version of console.log().

JS vs Python

Booleans and missing values

Python capitalises boolean values and uses None for empty or missing values.

JavaScript
let passed = true;
let remarks = null;
Python
passed = True
remarks = None

print(passed)
print(remarks)

Python Organisation

Writing your own functions

Built-in functions are ready-made. def lets you create your own reusable tool.

Python
def calculate_average(numbers):
    total = sum(numbers)
    average = total / len(numbers)
    return average

scores = [72, 85, 60, 91]
print(calculate_average(scores))
